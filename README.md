
What is this project?

This is the first game I've ever made using HTML, CSS, JS. As you all know from my previous projects, I'm not that strong with JS, but I'm pretty happy that with this project I managed to noticeably reduce my AI usage compared to my last one.

So, what does it do?

It's a new game where you're stacking a full pancake tower and trying to line them up on top of each other (you'll get it once you try the game, 
I'm not great at explaining). The game has scores, and when you lose it shows you the highest score you got and what that score was.

I started with HTML and CSS, did them well, then did the JavaScript well too, but there were a few things that tripped me up — AI helped me with them, 
though not by much. When I finished coding and was checking through it,
I found a few mistakes in the HTML and fixed them. I also ran into a problem I ended up solving with AI, because honestly there was no solution I could find and I genuinely didn't know what to do — the game's fonts just weren't showing up. 
I opened F12, saw the issue, sent it to the AI, and it fixed it through my own CSS code after some adjustment, plus a few small style tweaks from it — not much at all — and I added them in. 
I made a specific commit right before AI's CSS adjustments to show you there were no real differences before and after AI — it just fixed the font issue, adjusted the color shades, and made a couple of small tweaks, nothing more.

So we could say:





I did 95% of the HTML completely without AI



CSS: I did 85%, AI helped a bit



JS: my AI usage dropped noticeably compared to the last project — I did 75% of the JavaScript without AI

We could say I finished 75% of this project without AI.

Looking ahead, I'm thinking about adding a backend and a database to the game, and building a server with rooms so friends can play together — everyone joins the same room and sees who among them gets the highest score. There are a lot of changes I'm planning to make.

Finally, I'd like to thank Hack Club for their support, and thanks to the reviewers for their time.
